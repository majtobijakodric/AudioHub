# AudioHub

This is a free and open source webapp for listening music. It mostly written in TypeScript and ran in nodeJS.


## How to install?
To use these commands you need to have **nodeJS** and **git** installed on your system! 

***(Windows)***
Install nodeJS from https://nodejs.org/en/

***(Linux)*** You should know how to do this


```bash
# Clone the repository
git clone https://github.com/majtobijakodric/audiohub.git

# Install TypeScript
npm install typescript

# Install dependencies
npm install
```

## Run the app
This will run the nodemon module that watches for any changes in ***src*** folder and refresh the page if any are made.
```bash
npm run dev 
```
